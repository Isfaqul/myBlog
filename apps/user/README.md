# User Frontend

This is the public side of my blog.

You can preview it [here](https://isfaq-blog.netlify.app)

It is built with React, TypeScript, Vite, and Tailwind CSS.

This is where visitors can:

- Read blog posts
- View individual articles
- Leave comments
- Learn more about me

The site talks to the backend API for all data. It does not store anything on its own.
