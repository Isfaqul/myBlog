# Admin Frontend

This is the admin side of my blog.

It is built with React, TypeScript, Vite, and Tailwind CSS.

This app is used to:

- Log in as an admin
- Create new blog posts
- Edit existing posts
- Publish or unpublish posts
- Manage comments

It connects to the same backend API as the public site, but only allows access to protected routes for authenticated admin users.

The goal of this app is to keep content management simple and straightforward.
